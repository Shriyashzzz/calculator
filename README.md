# Calcultor

classic calculator project haha, had to do it


Built with HTML, CSS, and vanilla JavaScript — no frameworks or libraries.


Take a look at it live : https://shriyashzzz.github.io/calculator/

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal support
- Clear and delete functionality
- SUpports some keyboard input, Enter, & Backspace.

## Purpose

Built to practice and reinforce core concepts including DOM manipulation, JavaScript array methods, and refresh memoryon  CSS layout.

## Getting Started

Clone the repo and open `index.html` in your browser.

```bash
git clone git@github.com:Shriyashzzz/calculator.git
```
